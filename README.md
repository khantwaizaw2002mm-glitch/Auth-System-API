# Node.js Todo and Movie CRUD API

A comprehensive RESTful API built with Node.js, Express, and MongoDB that manages both Todo tasks and a Movie database with full CRUD functionality.

## Features

### 🎬 Movie Management (Full CRUD)
* Create: Add new movies with details (Title, Director, etc.).
* Read: Fetch all movies or search for specific movie details.
* Update: Edit existing movie information and reviews.
* Delete: Remove movies from the database.

### ✅ Todo Management
* Complete task management system to Create, Read, Update, and Delete todo items.

### 🛠 Architecture
* Implemented using the Controller-Service-Model design pattern for clean and maintainable code.

## Tech Stack
* Runtime: Node.js
* Framework: Express.js
* Database: MongoDB (via Mongoose)
